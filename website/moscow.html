MoSCow Requirements



Must Have
  • Submit FOI requests through the chatbot
  • Provide a chatbot endpoint that can be hooked up with some chat platform
Should Have
  • Categorize new requests
  • Provide information on previously answered questions
  • Access bot through services such as skype or facebook messenger
  • Analyse how similar the request is to previously asked questions

  • Gather statistics on the conversations
  • Answer new questions from knowledge gathered from previous questions
  • Request Validation
