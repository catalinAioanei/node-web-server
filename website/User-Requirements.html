User Requirements



Must Have
  • Target at current residents in Camden or prospective residents who are looking to move into Camden
  • Users group: Adults over 18
  • Implemented as a chatbot interface on the Camden Council webpage
Should Have
  • Include the use of buttons for easy response from user
  • Include a spell checker to allow the chatbot to understand what the user means if there is a typo in the question
  • Provide suggestions using keywords if the chatbot does not fully understand the question
  • Provide a reply in simple plain English for easy understanding
  • Allow the chatbot to be used on mobile browser
Could have
  • Include the chatbot in the Camden Council TTS service to improve accessibility
  • Store questions that the chatbot is unable to answer for analysis
